# ⚖️ Kelly AI – Legal Assistant Chatbot

## Overview

Kelly AI is an intelligent legal assistance platform designed to help users understand legal concepts, IPC sections, case summaries, and general legal information through an interactive AI-powered chatbot.

The application integrates Google's Gemini AI API to provide real-time responses to legal queries. Users can also upload images and interact with the AI for enhanced assistance. The platform includes secure user authentication, account creation, and database integration for managing user accounts.

## How It Works

1. User registers or logs in.
2. User enters a legal query.
3. Query is sent to Google Gemini API.
4. Gemini processes the request.
5. AI generates a legal response.
6. Response is displayed in the chatbot interface.

For image-based queries:

1. User uploads an image.
2. Image is converted to Base64 format.
3. Image and prompt are sent to Gemini API.
4. Gemini analyzes the content.
5. AI-generated response is returned to the user.
